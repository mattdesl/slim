--- TWL themes by davedes (Ver 1) ---

=== Chutzpah Theme ===
UI designed by Morgan Allan Knutson:
http://designmoo.com/2435/chutzpah-user-interface-kit/

Font is PT Sans:
http://www.paratype.com/public/ 

= TWL Info =
CSS font families: default, italic, link
Cursors: cursor_pointer, cursor_text
Themes: label, button, togglebutton, checkbox, radiobutton, editfield, vscrollbar, 
        hscrollbar, listbox, combobox, scrollpane, tooltipwindow, textarea, panel

The "previewwidgets" theme can be removed as it's mainly for testing.
